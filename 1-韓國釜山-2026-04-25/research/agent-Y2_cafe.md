## Y2-cafe: 釜山在地咖啡 5 家（非田浦）

> 鎖定 25-35 在地年輕人 + IG / 小紅書當紅店家，各區散布：西면 / 廣안리 / 海雲台 / 影島 / 기장-송정 各 1 家。**不收田浦**（有專門 agent）。

### 1. 西면 — 카페벽돌 (Cafe Byeokdol / 紅磚咖啡)

- **中韓文名**：카페벽돌（直譯「磚塊咖啡」）
- **地區**：釜山진구 西面（서면）
- **地址**：부산광역시 부산진구 서면로 46（2F-5F + 頂樓 rooftop）
- **Google Maps**：https://maps.google.com/?q=카페벽돌+서면로+46
- **Instagram**：[@cafe_byukdol](https://www.instagram.com/cafe_byukdol/)
- **小紅書關鍵字**：搜「釜山 카페벽돌 / 西面紅磚咖啡」就有

**招牌 + 價位（韓元）**

- 招牌 espresso 系列（店家自己 IG bio 強調 espresso）：單份 espresso ~3,500 / 美式 4,500 / 拿鐵 5,500
- 甜點 + 飲品套餐：平日 11:30 開店 ~ 14:00 早鳥組合 8 折
- 整體客單價 ~7,000-12,000 KRW（約台幣 170-290）

**為什麼年輕人愛 / 朋友團友善**

- 整棟 4 層樓 + 頂樓露台的紅磚倉庫風建築，**從外觀就辨識度爆表**，韓國 MZ 世代「벽돌 감성」（紅磚氛圍）的代表店之一
- 3F 有面對市區的 bar 吧台座、4F 有沙發區、頂樓露台可抽菸 + 看西面夜景 → 朋友團多人去**完全不擠**，每層樓功能不同可以分散
- 西面地鐵站走 5 分鐘，逛完樂天百貨 / 西面市場順路殺進去坐
- 夜貓友善：營業到 22:00（最後點餐 21:00），晚餐後續攤喝咖啡聊天剛好

### 2. 廣안리 — 블루코스모스 (Blue Cosmos)

- **中韓文名**：블루코스모스（Blue Cosmos）
- **地區**：수영구 광안리（廣安里）
- **地址**：부산광역시 수영구 광안로35번길 71, 1층
- **Google Maps**：https://maps.google.com/?q=블루코스모스+광안리
- **Instagram / 小紅書**：IG 搜「blue cosmos busan / 블루코스모스」；小紅書搜「釜山 廣安里 藍宇宙」「廣安里 角色咖啡廳」

**招牌 + 價位（韓元）**

- 招牌：超大份量「코스모스 케이크 (cosmos cake)」、藍色系漸層飲品、店內限定角色周邊
- 飲品 5,500-7,500 / 蛋糕 7,000-9,000 / 客單價 ~12,000-16,000 KRW

**為什麼年輕人愛 / 朋友團友善**

- 整間店塞滿可愛 character plush + 復古玩具 + pastel 配色道具，**每個角落都是拍照點**，IG 上釜山 MZ 必拍
- 空間夠大，朋友團去不會卡到別桌拍照
- 距離廣安里海邊步行 5 分鐘，**拍完海邊接著拍咖啡廳**動線順
- 重點是甜點走「奶油爆量、配色衝擊」路線，小紅書「韓國 인생샷（人生美照）」風格剛好打中華語旅客

### 3. 海雲台（青沙浦）— LIBERTA 리베르타 청사포점

- **中韓文名**：LIBERTA（리베르타）青沙浦店
- **地區**：海雲台구 청사포（青沙浦，海雲台天空膠囊列車終點）
- **地址**：부산광역시 해운대구 청사포로 137
- **Google Maps**：https://maps.google.com/?q=LIBERTA+청사포로+137
- **Instagram**：[@liberta___cheongsapo](https://www.instagram.com/liberta___cheongsapo/)
- **小紅書關鍵字**：「釜山 LIBERTA / 青沙浦 海景咖啡」

**招牌 + 價位（韓元）**

- 一樓是大型 bakery，現烤可頌、巴斯克乳酪、鹽可頌、栗子塔等十幾款，**烘焙是真招牌**
- 麵包 / 甜點 4,500-7,500 / 飲品 6,000-7,500 / 客單價 ~13,000-17,000 KRW
- 營業 09:00-19:30，**早午茶時段尤其推**

**為什麼年輕人愛 / 朋友團友善**

- 三層樓整面落地窗對著青沙浦海岸 + 廣安大橋遠景，**每層都有海景座位**
- 從海雲台搭天空膠囊列車（Sky Capsule）到青沙浦站走出來就到，**搭可愛粉藍膠囊 + 海景咖啡的 combo 是釜山 MZ / 小紅書最熱門路線**
- 一樓選麵包 → 樓上找位置的動線讓朋友團「各拿各的、坐一起分食」很順
- 開到 19:30 可看夕陽染海

### 4. 影島 — Having Moment 해빙모먼트

- **中韓文名**：해빙모먼트（Having Moment）
- **地區**：영도구 흰여울문화마을（白淺灘文化村）
- **地址**：부산광역시 영도구 흰여울길 일대 (白淺灘文化村上層主路旁)
- **Google Maps**：https://maps.google.com/?q=해빙모먼트+흰여울문화마을
- **Instagram / 小紅書**：IG 搜「havingmoment / 해빙모먼트」；小紅書搜「釜山 白淺灘 黑芝麻拿鐵」「영도 having moment」

**招牌 + 價位（韓元）**

- **招牌：흑임자 크림 라떼（黑芝麻奶油拿鐵）** — 綿密黑芝麻奶蓋 + 微鹹尾韻，是這家在小紅書 / IG 上爆紅的單品
- 也有抹茶奶油拿鐵、季節水果氣泡飲
- 飲品 6,500-8,500 / 簡單甜點 5,500-7,000 / 客單價 ~11,000-15,000 KRW

**為什麼年輕人愛 / 朋友團友善**

- 站門口正對南港大橋 → 松島的廣闊海岸線，**白淺灘文化村咖啡廳裡視野最開的之一**
- 黑芝麻奶蓋拿鐵的視覺（深灰漸層）+ 海景背景在 IG / 小紅書是釜山招牌構圖之一
- 白淺灘文化村本身就是步行散策動線，朋友團「散步 → 進來歇腿喝杯 → 繼續走」剛好
- 從南浦地鐵站搭公車約 15 分鐘，回程也能順遊太宗台 / 깡깡이藝術村

### 5. 기장 — WAVEON COFFEE 웨이브온커피

- **中韓文名**：웨이브온커피（Wave On Coffee）
- **地區**：기장군 장안읍（機張郡，釜山東北郊）
- **地址**：부산광역시 기장군 장안읍 해맞이로 286
- **Google Maps**：https://maps.google.com/?q=웨이브온커피+해맞이로+286
- **Instagram / 小紅書 / TikTok**：IG 搜「waveon coffee / 웨이브온커피」；小紅書搜「釜山 Waveon / 機張 玻璃屋咖啡」；TikTok 大量影片用「#waveoncoffee」

**招牌 + 價位（韓元）**

- 招牌不是飲品而是「**空間本身**」 — 韓國知名建築師郭熙秀（곽희수）操刀，曾獲韓國建築文化大獎
- 美式 5,500 / 拿鐵 6,500 / 季節 signature 飲品 7,500-9,000 / 蛋糕 7,000 起
- 客單價 ~13,000-17,000 KRW
- 營業 10:00-24:00（最後點餐 23:00），**夜景也好拍**

**為什麼年輕人愛 / 朋友團友善**

- 三層樓 + 頂樓露台的全玻璃帷幕海景建築，**360 度看海**，釜山所有海景咖啡廳裡空間最大、視野最開的
- IG 上「釜山必去」reels 流量最高的咖啡廳之一，韓國 MZ + 華語旅客都認
- 空間超大 → 朋友團 4-6 人不用排隊也能找到位
- 缺點：在機張，從海雲台/西面要搭計程車或公車約 40 分鐘 — **適合排「半日機張行程」配 오시리아（Osiria 海岸）的當天**，不適合臨時插

### 收尾建議

如果只能挑 1-2 家插進 Day 2 / Day 3 空檔，我推這個組合：

- **Day 2 推 LIBERTA 청사포店**：與「海雲台 → 搭天空膠囊列車到青沙浦」這條釜山 MZ / 小紅書經典路線完全綁定，下午茶剛好接夕陽。三層樓 + 海景 + bakery 對朋友團每個人需求（吃麵包的、純喝咖啡的、想拍照的）一網打盡，動線最順、CP 值最高。
- **Day 3 推 Having Moment（影島）**：黑芝麻奶油拿鐵是這趟最容易做出記憶點 + 在 IG / 小紅書上「我去過釜山」最具識別度的單品。配白淺灘文化村半日散策。如 Day 3 排機張線則改成 **Waveon Coffee** 替代（同一天進機張順路、避免拉車兩次）。
- 西面 카페벽돌 / 廣안리 블루코스모스 都是「逛街順路型」，沒空檔就跳過、不影響整體行程精彩度。
