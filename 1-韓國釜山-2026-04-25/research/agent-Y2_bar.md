## Y2-bar: 釜山年輕 Bar / 居酒屋 4 家

> 朋友團小酌路線（避開老派酒店 / Room Salon），主打 MZ 世代 IG 打卡 + 氛圍店。一區一家：西면、廣안리、田浦、수영 / 海雲台。

### 1. 西면 — 우아시 (Our Beautiful Time / 우리들의 아름다운 시간)

- **中韓文名**：우아시（우리들의 아름다운 시간，意「我們的美好時光」）
- **地區**：釜山鎮區 西면（Seomyeon），東川路 85 巷
- **地址**：부산 부산진구 동천로85번길 22, 2층（2 樓）
- **Google Maps**：搜尋「우아시 서면」或 `https://maps.google.com/?q=우아시+서면+부산`
- **Instagram**：[@our_beautiful__time](https://www.instagram.com/our_beautiful__time/)（西면술집 / 西면안주 / 西면핫플 自封標籤）
- **TikTok / 小紅書**：小紅書搜「釜山西面 우아시」可看到台港女子團打卡；TikTok hashtag `#서면술집` 出現率高

**主打酒款 + 價位**
- 韓式創意 highball（柚子 / 草莓 highball）約 ₩9,000–11,000
- 日式 sour、生啤 ₩6,000 起
- 招牌下酒菜：明太子玉子燒、和牛炙燒壽司、鹹酥雞拼盤 ₩18,000–28,000
- 人均約 ₩30,000–40,000（含 1.5 杯酒 + 2 道下酒菜）

**為什麼朋友團愛 / 氛圍 / 女生團友善度**
- 暖橘燈 + 木質吧台 + 低天花板的「居酒屋包圍感」，IG 出片率高（多數打卡角度都在 2 樓窗邊）
- 17:00–04:00 連夜開到天亮，西面站 2 號出口走 3 分鐘，散場可直接搭計程車
- 桌距偏近但不擠，**4 人桌**最舒服；女生團友善（很多女子 2-4 人組客群、店員會主動幫拍照）
- **避雷**：週五六 21:00 後常滿，建議 19:00 前到或 IG 私訊預約

### 2. 廣안리 — 광안 부단 (Gwangan Budan)

- **中韓文名**：광안 부단（廣安 Budan，自稱 emotional dining pub）
- **地區**：水營區 廣안리 海邊（廣安里海水浴場第一排，正對廣安大橋）
- **地址**：부산 수영구 광안해변로 195, 2-3층（2、3 樓兩層樓店）
- **Google Maps**：搜尋「광안 부단」或 `https://maps.google.com/?q=광안부단+광안리`
- **Instagram**：[@gwanganbudan](https://www.instagram.com/gwanganbudan/)（光是「광안리 핫플」hashtag 就有上百萬貼文）
- **小紅書 / TikTok**：小紅書搜「광안리 부단」「廣安里 Budan」可見大量台港陸客打卡；TikTok `#광안리술집` 出現率第一

**主打酒款 + 價位**
- 招牌：**廣安大橋夜景 highball / 水果調酒** ₩12,000–14,000（杯緣裝飾很 IG）
- 生啤 / 燒酎雞尾酒 ₩7,000–9,000
- 下酒菜走「emotional pub」路線：松露薯條、烤起司海鮮拼盤、和牛 tataki ₩22,000–35,000
- 人均約 ₩40,000–55,000（含調酒 + 共享 2 道菜）

**為什麼朋友團愛 / 氛圍 / 女生團友善度**
- **2、3 樓全是落地窗正對廣安大橋**，晚上 8 點橋燈秀（燈光秀通常 20:00 / 21:00 / 22:00 各一場）配調酒是釜山必吃畫面
- IG 主題店裝潢：暖燈 + 海景 + 自帶燈牆，朋友團合照不用後製
- 適合 4 人桌，靠窗位需提前 IG DM 預約（週末必訂）
- **女生團超友善**：客群 7 成以上是 20-30 歲女子組合，店員拍照技術好
- **避雷**：週五六晚 19:00 後沒預約進不去；旺季（夏天 / 煙火節）排隊 1 小時起跳

### 3. 田浦 — 드므부산 (Deumeu Busan / 드므)

- **中韓文名**：드므부산（Deumeu Busan，韓文「드므」是傳統陶甕的意思，店內裝潢有甕的元素）
- **地區**：釜山鎮區 田浦洞（전리단길 / 전포 카페거리 範圍內）
- **地址**：부산 부산진구 전포대로 209번길 一帶（전포역 8 번 출구 步行 3 分鐘）
- **Google Maps**：搜尋「드므부산」或 `https://maps.google.com/?q=드므+전포`
- **Instagram**：店家自媒體在 `@jeonpo_zip`（전포술집 集合帳號）有頻繁曝光；可搜 `드므부산` IG hashtag
- **小紅書 / TikTok**：小紅書搜「전포 와인바」「田浦 wine bar」必出現；TikTok `#전포술집` 多次入榜

**主打酒款 + 價位**
- 自然酒（natural wine）杯酒 ₩9,000–13,000、瓶裝 ₩45,000–90,000（韓國 wine bar 中算友善價）
- 韓式 makgeolli（瑪格利酒）跟 wine 混搭調飲
- 招牌：**Burrata 拼盤 + 烤蘑菇 + 韓牛 carpaccio** ₩18,000–28,000
- 人均約 ₩35,000–50,000（含 1 杯酒 + 共享 2 道菜）

**為什麼朋友團愛 / 氛圍 / 女生團友善度**
- 田浦 = 釜山版「東京中目黑 / 台北赤峰街」氛圍，**整條街全是小巧 wine bar / 酒場**，飯後散步酒吧巡禮
- 드므 走「灰泥牆 + 暖燈 + 木桌」極簡風，IG 拍起來像首爾 / 東京店
- 週日到一 17:00–24:00、週五六 17:00–01:00，**無公休**，朋友團晚到不怕撲空
- **女生團特別友善**：客單偏年輕女子組，店主 / 店員態度好，吧台座可以單獨 2 人或 4 人圍小桌
- **避雷**：座位偏少（約 20 席），週五六 20:00 後常滿，建議 19:00 前到

### 4. 수영 / 海雲台 — 밀락더마켓 (Millac the Market) — MTS 야시장 + NAKED APE Bar

- **中韓文名**：밀락더마켓（Millac the Market，複合文化空間，內含野市場 MTS / 手釀啤酒 Bar / 多間餐酒館，朋友團 4 人各點各的最 OK）
- **地區**：水營區 民樂洞 海邊（水變路，從 廣안리 海水浴場走過去 10 分鐘，正對廣安大橋另一面）
- **地址**：부산 수영구 민락수변로17번길 56
- **Google Maps**：搜尋「밀락더마켓」或 `https://maps.google.com/?q=밀락더마켓+민락수변로`
- **Instagram**：[@millac_the_market_official](https://www.instagram.com/millac_the_market_official/) / 夜市專屬 [@mts_nightmarket](https://www.instagram.com/mts_nightmarket/)
- **小紅書 / TikTok**：小紅書搜「밀락더마켓」「Millac 釜山」必有大量 vlog；TikTok `#밀락더마켓` 是釜山近兩年最熱 hashtag 之一

**主打酒款 + 價位**
- **NAKED APE 手釀啤酒**（英國 brewer Paul Edwards 釀造）：IPA / Stout / Lager 各 ₩9,000–13,000
- MTS 야시장內各攤位調酒 ₩6,000–10,000、生啤 ₩5,000–7,000（價格比西면 / 田浦友善）
- 配酒小食：手作漢堡、辣炒年糕、薯條、丼飯 ₩8,000–18,000
- 人均約 ₩25,000–35,000（含 2 杯酒 + 一份小食，4 區最便宜）

**為什麼朋友團愛 / 氛圍 / 女生團友善度**
- **複合空間 = 4 人各拿各的 = 不用為了選哪家吵架**：有人想喝精釀、有人想吃宵夜、有人想拍夜市照，全部一站搞定
- MTS 야시장 19:00–凌晨 03:00 營業（最後點餐 02:00），是釜山少數真的撐到天亮的合法夜市
- **海景 + 燈泡串 + 工業風貨櫃**，IG 出片率破表，朋友團隨便拍都像 vlog
- 露天 + 室內都有，4 月底釜山溫度約 18-20°C，**夜晚海邊偏涼，建議帶薄外套**
- **女生團友善度滿分**：人潮極多 + 開放空間 + 有保全，是最安全的深夜選項；廁所動線 OK
- **避雷**：週末（4/25 是週六、4/26 是週日）人爆多，建議 19:00 前到佔位；煙火 / 表演時段（檢查 IG 公告）人潮再翻倍

## 收尾建議

**Day 1 收尾建議：西면 우아시**。第一天通常飛機落地、行李丟飯店後想找個能坐下來、暖燈包圍、不需要走太多路就能到的地方暖身。우아시 在西면站旁、開到凌晨 4 點，朋友團剛抵達想 catch up 聊天最合適；菜單也涵蓋輕食到下酒菜，不會吃太飽。

**Day 2 收尾建議：水營 밀락더마켓 + 廣安里散步**。第二天通常已經跑了甘川 / 海雲台等景點體力下降，밀락더마켓的「自助式 + 開放空間 + 海景 + 平價手釀啤酒」最不費力，4 個人各拿各的、不用討論吃什麼，配廣안대교夜景；行程結束想直接離開或繼續往廣안리海邊散步都方便。

**Day 3 收尾建議：田浦 드므부산 或 廣안리 광안 부단（看心情選一）**。最後一晚走「儀式感 + 拍美照」路線：想要自然酒 + 街區漫遊感選 **드므부산**（田浦巷弄文青夜），想要海景 + IG 大景做完美收尾選 **광안 부단**（最後一晚的廣安大橋夜燈秀絕配）。如果朋友團裡有人沒去過廣안리，強烈推 광안 부단；如果第二天已經去過 밀락더마켓，第三天就改去田浦避免重複海景。
