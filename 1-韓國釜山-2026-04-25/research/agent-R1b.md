# Agent R1b — 慶州 / 大邱 / 夜生活 / 韓服 研究報告

## A3 慶州歷史古蹟 + 拍照重點 Top 6

### 1. 佛國寺（불국사）
- **地址**：慶尚北道慶州市佛國路 385（경상북도 경주시 불국로 385）
- **Google Maps**：https://www.google.com/maps/search/불국사
- **為什麼推薦 / IG 角度**：1995 年聯合國世界文化遺產，新羅佛教文化的代表，慶州必訪 No.1。多寶塔、釋迦塔、青雲橋、白雲橋是經典構圖；春天櫻花、秋天楓紅，IG 上「불국사 벚꽃」非常熱門。可拍韓幣 10 元上的多寶塔對照圖。
- **建議停留**：1.5–2 小時
- **入場費**：成人 6,000 韓元（約 NT$140）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=불국사

### 2. 石窟庵（석굴암）
- **地址**：慶尚北道慶州市佛國路 873-243（경상북도 경주시 불국로 873-243）
- **Google Maps**：https://www.google.com/maps/search/석굴암
- **為什麼推薦 / IG 角度**：與佛國寺同列世界文化遺產，吐含山山腰的人造石窟，東方最大如來坐像（內部禁止拍照），但通往石窟庵的山徑、東海日出眺望點是攝影師最愛。建議與佛國寺安排同日。
- **建議停留**：1–1.5 小時
- **入場費**：成人 6,000 韓元
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=석굴암

### 3. 瞻星台（첨성대）
- **地址**：慶尚北道慶州市仁旺洞 839-1（경상북도 경주시 인왕동 839-1）
- **Google Maps**：https://www.google.com/maps/search/첨성대
- **為什麼推薦 / IG 角度**：建於新羅善德女王時期（632–647 年），東洋現存最古老的天文台，27 層石筒造型獨特。秋天周邊的粉黛亂子草（핑크뮬리）花海是 IG 爆款；春天則有油菜花、黃花。低角度仰拍 + 花海前景最殺。
- **建議停留**：30–45 分鐘
- **入場費**：免費（開放式園區）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=첨성대

### 4. 皇理團路（황리단길）
- **地址**：慶尚北道慶州市皇南洞一帶（경상북도 경주시 황남동）
- **Google Maps**：https://www.google.com/maps/search/황리단길
- **為什麼推薦 / IG 角度**：被稱為「大型益善洞」的傳統韓屋商店街，1960–70 年代老建築改建的咖啡廳、寫真館、韓服店林立。穿韓服走在巷弄間是最熱門 IG 場景；慶州星巴克（韓屋造型）必拍；十元餅、慶州麵包是必吃。
- **建議停留**：2–3 小時（含咖啡 / 用餐）
- **入場費**：免費
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=황리단길

### 5. 大陵苑 / 天馬塚（대릉원 / 천마총）
- **地址**：慶尚北道慶州市鷄林路 9（경상북도 경주시 계림로 9）
- **Google Maps**：https://www.google.com/maps/search/대릉원
- **為什麼推薦 / IG 角度**：23 座新羅王陵聚集的古墳公園，世界文化遺產。天馬塚是唯一可入內參觀的古墳。最著名 IG 拍照點是「兩座古墳間的木蓮樹」（春天 4 月開花，圓墳 + 白色木蓮的對比是慶州春季代表畫面）；秋天楓葉同樣壯觀。
- **建議停留**：1–1.5 小時
- **入場費**：成人 3,000 韓元（開放至 22:00）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=대릉원

### 6. 東宮與月池（동궁과 월지 / 雁鴨池）
- **地址**：慶尚北道慶州市源花路 102（경상북도 경주시 원화로 102）
- **Google Maps**：https://www.google.com/maps/search/동궁과월지
- **為什麼推薦 / IG 角度**：慶州夜景 No.1。新羅王宮離宮遺跡，夜晚點燈後宮殿倒映在月池水面，是韓國最美夜景之一。建議天黑後（4 月約 19:30 後）入場，水鏡反射構圖必拍。三腳架 / 慢快門效果最佳。
- **建議停留**：1–1.5 小時
- **入場費**：成人 3,000 韓元（開放至 22:00）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=동궁과월지

## A4 大邱必訪景點 Top 6

### 1. 西門市場（서문시장）
- **地址**：大邱廣域市中區大新洞 115-378（대구광역시 중구 큰장로 26길 45）
- **Google Maps**：https://www.google.com/maps/search/서문시장
- **為什麼推薦 / IG 角度**：大邱最大傳統市場 + 韓國知名夜市。白天賣生鮮百貨，晚上 7 點後變身夜市，攤販林立、煙火氣十足。必吃刀削麵、扁餃子、五花肉蔬菜捲、烤肥腸；夜市霓虹招牌與人潮是 IG 紀實感首選。
- **建議停留**：夜市 1.5–2 小時
- **入場費**：免費（夜市攤販每樣 3,000–8,000 韓元）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=서문시장

### 2. 金光石街（김광석 다시 그리기 길 / 金光石再描繪街）
- **地址**：大邱廣域市中區大鳳洞 252-1（대구광역시 중구 달구벌대로 450길 일대）
- **Google Maps**：https://www.google.com/maps/search/김광석길
- **為什麼推薦 / IG 角度**：紀念已故傳奇民謠歌手金光石的壁畫小巷，全長約 350m。歌詞、彩繪、銅像、吉他雕塑充滿文藝氣息，2019–2022 連續入選韓國觀光 100 選。每面壁畫都是 IG 取景點，文青風必拍。
- **建議停留**：1 小時
- **入場費**：免費
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=김광석길

### 3. 近代文化巷 / 近代胡同（근대골목 / 청라언덕）
- **地址**：大邱廣域市中區桂山洞 / 青蘿丘陵（대구광역시 중구 청라언덕길）
- **Google Maps**：https://www.google.com/maps/search/청라언덕
- **為什麼推薦 / IG 角度**：大邱「近代文化路 第二街」，串連青蘿丘陵、桂山聖堂、3·1 萬歲運動路（90 階階梯）、宣教士住宅。日式 + 西式古建築混搭，是大邱最具歷史氛圍的散步路線。桂山聖堂的紅磚哥德式外觀必拍。
- **建議停留**：1.5–2 小時
- **入場費**：免費
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=근대골목

### 4. 東城路商圈（동성로）
- **地址**：大邱廣域市中區東城路一帶（대구광역시 중구 동성로）
- **Google Maps**：https://www.google.com/maps/search/동성로
- **為什麼推薦 / IG 角度**：大邱版「明洞 + 弘大」，年輕人聚集地，地鐵 1 號線中央路站、半月堂站。Spark Land 摩天輪、大型 LED 螢幕、街頭表演（버스킹）、潮牌店 + 美妝 + 韓式炸雞應有盡有。夜晚霓虹燈最有都市感。
- **建議停留**：3–4 小時（含晚餐 / 購物）
- **入場費**：免費
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=동성로

### 5. 八公山（팔공산）
- **地址**：大邱廣域市東區八公山纜車（대구광역시 동구 팔공산로185길 51）
- **Google Maps**：https://www.google.com/maps/search/팔공산
- **為什麼推薦 / IG 角度**：大邱代表性山脈，纜車約 8 分鐘上山，俯瞰整片紅黃楓葉景觀。山上有觀景台、步道、咖啡廳。秋天楓葉、春天櫻花是大邱近郊一日遊首選。桐華寺（동화사）也在山腳下，可順遊。

- **入場費**：纜車來回 12,000 韓元
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=팔공산

### 6. 大邱三星獅棒球場（대구삼성라이온즈파크 / 라팍）
- **地址**：大邱廣域市壽城區延號洞（대구광역시 수성구 야구전설로 1）

## A5 夜生活與夜景

### 1. 西面酒吧街 / 西面 Nightlife（서면 술집거리）
- **地址**：釜山廣域市釜山鎮區西面一帶（부산광역시 부산진구 서면）
- **Google Maps**：https://www.google.com/maps/search/서면+술집거리
- **為什麼推薦 / IG 角度**：釜山最熱鬧的市中心夜生活區，西面地鐵站 1、2、4、6、7 號出口周邊，集中酒吧、Pocha（포차 / 韓式帳篷酒吧）、夜店、24h 烤肉。年輕人聚集、密度極高，適合「釜山夜晚第一站」。捷運 1、2 號線交會非常方便。霓虹街景、路邊攤桌椅是 IG 紀實風首選。
- **建議停留**：3–4 小時
- **入場費**：免費（人均消費 30,000–50,000 韓元）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=서면술집

### 2. 廣安里 The Bay 101（더베이101）
- **地址**：釜山廣域市海雲臺區佑洞 747-7（부산광역시 해운대구 동백로 52）

- **入場費**：免費（炸魚薯條套餐約 25,000 韓元、夜遊艇約 30,000 韓元）
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=더베이101

### 3. 廣安里海水浴場 + 廣安大橋（광안리해수욕장 + 광안대교）

- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=광안리

### 4. 大邱東城路夜景（동성로 야경）

## A6 韓服體驗 / 寫真館

> 慶州皇理團路一帶是慶州韓服體驗一級戰區，價格普遍便宜（1 小時 10,000 韓元起）、款式新潮，超適合搭配大陵苑、瞻星台、東宮與月池外拍。年輕人最愛「款式偏現代改良 + 髮型造型 + 拍照棚」這種一條龍服務。

### 1. 慶州韓服正中央（경주한복판 / Gyeongju Hanbok Junggang）
- **地址**：慶尚北道慶州市鮑石路（皇理團路中段，경상북도 경주시 포석로 1050 일대）
- **Google Maps**：https://www.google.com/maps/search/경주한복판
- **為什麼推薦 / IG 角度**：皇理團路最中央位置，**款式多 + 跟得上潮流**為強項。包含兒童、男性、夫妻情侶款，材質花色齊全（春夏輕薄款 / 秋冬絨布款），季節限定花色經常更新。老闆會幫忙做簡單髮型、提供髮飾配件。**體驗結束後可手機連線印照片**（適合做紀念）。地點走 5 分鐘到大陵苑、10 分鐘到瞻星台，外拍動線最順。
- **價位**：
  - 1 小時：10,000 韓元（約 NT$230）
  - 3 小時：20,000 韓元（約 NT$460）
  - 4 小時：25,000 韓元
  - 一日券：30,000 韓元
- **建議停留**：換裝 + 髮型 30 分鐘 + 外拍 2–3 小時
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=경주한복판

### 2. 皇南洋裝店（황남양장점 / Hwangnam Yangjangjeom）
- **地址**：慶尚北道慶州市皇理團路（경상북도 경주시 포석로）
- **Google Maps**：https://www.google.com/maps/search/황남양장점
- **為什麼推薦 / IG 角度**：**多元服飾路線**強店——除了傳統韓服，還有「**開化期古典洋服**」（1900 年代初期復古洋裝）+ **復古校服**（70–80 年代制服）+ 帽子配件 + 室內主題攝影棚。年輕人 IG 最愛「開化期洋裝 + 慶州古蹟」的反差感。可室內拍 + 外拍整個慶州歷史區。
- **價位**：韓服 1 小時 10,000 韓元；古典洋服 / 校服約 15,000–25,000 韓元（依套裝）；室內攝影棚加價 5,000 韓元
- **營業**：10:00–18:30
- **建議停留**：2–3 小時
- **IG 搜尋**：https://www.instagram.com/explore/search/keyword/?q=황남양장점

> 預約建議：旺季（春櫻 4 月、秋楓 10–11 月）週末熱門款很快租完，建議至少**前一天 KKday / Klook 預約**或現場上午 10 點到。穿韓服進大陵苑、佛國寺、東宮與月池等景點可**免門票**（皇理團路韓服店通常會發證明）。

## 收尾：慶州 + 大邱必去 5 大（按優先排）

**1. 東宮與月池夜景（慶州）** → 韓國最美夜景之一，宮殿水鏡反射，慶州行程必排夜景檔  **2. 佛國寺 + 石窟庵（慶州）** → 世界文化遺產雙打，慶州精神象徵，半日綁定行程  **3. 皇理團路 + 大陵苑韓服外拍（慶州）** → 韓屋商店街 + 古墳公園木蓮樹 + 韓服 = 慶州 IG 三大殺手鐧  **4. 西門市場夜市（大邱）** → 大邱必體驗，刀削麵 + 烤肥腸 + 扁餃子煙火氣最強  **5. 金光石街 + 近代文化巷（大邱）** → 文青文化散步路線，半日可走完。八公山、大邱三星獅球場視時間 / 興趣加碼。
