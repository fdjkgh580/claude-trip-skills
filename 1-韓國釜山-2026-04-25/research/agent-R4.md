## D4. Check-in/out、行李寄放、WiFi、雙人 / 三人房政策

### Check-in / Check-out
- **標準**：Check-in 15:00、Check-out 11:00（多數）；連鎖商旅可能 14:00 / 12:00
- **早 Check-in**：通常無法，可加付房價 20%-50% 早入住，否則只能寄放行李
- **延遲 Check-out**：問櫃台，1-2 小時通常免費，超過要算半日房費
- **抵達前可先 email 通知 ETA**（特別是凌晨班機抵達），避免被當 no-show

### 行李寄放
- **飯店**：所有中價位飯店都免費寄放，最少寄到當日 22:00（部分 24h）
- **地鐵置物櫃**：T-Locker app 查詢首爾 / 釜山地鐵置物櫃，西面、海雲台、南浦洞、釜山站都有，小櫃 ₩2,000-3,000、大櫃 ₩4,000-6,000 / 4 小時
- **離境前一天**：飯店退房後可寄放到下午搭機，回來拿很安全

### WiFi
- 所有中價位飯店標配免費 WiFi，速度通常 50-100 Mbps，足夠視訊
- 多數飯店有公用電腦 / 商務中心
- 韓國公共 WiFi（地鐵、咖啡店、機場）很普及

### 雙人 / 三人房政策（朋友團 2-4 人重點）
- **韓國飯店三人房比日本好找**，但**不一定加床**，常常是「雙人房 + 額外 1 人 ₩20,000-40,000」，多收的不一定有床
- **想睡好** → 訂兩間雙人房 / 訂 Family Room / 訂公寓式（如 Urbanstay）
- **想省錢 + 不介意打地鋪 / 沙發** → 訂雙人房付加人費（事先寫信問 single bed extra ₩? per night）
- **韓屋民宿**：傳統地暖房（온돌房），打地鋪睡棉被，3-4 人一間反而正常
- **Booking 注意**：選房型時看「最大入住人數」，超過要在備註欄寫清楚並付加人費


## E1. 治安整體評估

- **整體**：韓國 2025 全球和平指數 (GPI) 排名約第 41，亞洲屬中上、優於東南亞多數國家
- **釜山**：全韓國治安最好的大城市之一，比首爾還安靜。觀光區人多、晚上 23:00 後仍有人潮
- **重大暴力犯罪**：對觀光客機率極低
- **常見問題**：① 計程車繞路 / 黑車 ② 餐廳 / 商家漫天叫價 ③ 紅燈區誤闖 ④ 酒醉糾紛
- **獨自夜歸**：西面、海雲台、南浦洞 24:00 前都安全；偏巷小路、釜山站後方避開
- **女生獨旅**：釜山被多家旅遊媒體列為亞洲女性友善城市之一，但仍要避免單獨深夜逛酒吧街後巷


## E2. 常見詐騙與扒竊高風險區

### 高風險區
- **釜山站後方「外國人觀光街」**：標牌寫 BAR，實際是紅燈區（中華料理店 / 練歌房 / 桑拿招攬皆要小心被漫天叫價）。**直接避開**
- **西面酒吧街深夜後巷**：拉客「漂亮姊姊」「免費喝一杯」幾乎都是高消費黑店，最低消費可達 ₩300,000+
- **甘川洞文化村小巷**：白天觀光區無問題，但有少數扒手在人多時下手，背包反背
- **札嘎其市場**：注意被亂塞海鮮 / 不問價就秤重，買前一定**先問價、用 papago 翻韓文**

### 計程車詐騙（最常見）
- **黑車（無 TAXI 燈、無黃車牌）**：完全不要上
- **個人計程車（車頂「개인」字）**：盡量避，比公司計程車（橘 / 綠 / 銀）易出狀況
- **下車一定拿收據**（韓文 영수증 yeongsujeung）：上面有車牌 + 時間，事後申訴用
- **強烈推薦用 Kakao T 叫車**：跳表透明、自動算錢、不會被繞路，且 app 內可投訴
- **Uber Korea**：司機可能在乘客上車前就按「開始行程」偷跑里程，要用 PIN 碼確認功能

### 護照與財物
- 韓國治安整體好但**不是零扒竊**。地鐵 1 號線（西面 ↔ 南浦洞通勤車段）擁擠時段、夜店密集區是熱點
- 護照建議鎖飯店保險箱、隨身只帶**護照影本 + 一張信用卡 + 少量現金**

### 申訴管道
- **韓國觀光公社觀光投訴中心**：1330（24h，中文 OK），可協調退費
- **韓國旅遊不便諮詢處**：可線上填表 https://www.visitkorea.or.kr

## E4. 保險理賠須知（韓國醫院取得醫療收據、診斷書流程）

### 在韓國醫院就醫的步驟
1. **打 1330 找翻譯**（中文 OK）→ 請對方推薦有外籍人士窗口的醫院 + 預約
2. **帶護照** + 旅遊保險投保證明（截圖也可）到醫院櫃台 / 國際醫療中心
3. 填表（個人資訊、症狀、過敏史）→ 看診 → **當場結帳付現或刷卡**（韓國對外籍患者通常無健保價，但收費合理：感冒看診 ₩30,000-50,000；急診 ₩100,000-300,000）

### 必須跟醫院要的 3 樣文件（缺一不可）
1. **英文診斷證明書**（진단서 / Medical Certificate）：費用約 ₩20,000-30,000；務必請醫師寫**詳細**疾病名稱、發生時間、治療項目
2. **英文收據正本**（영수증 / Receipt）：費用約 ₩2,000-5,000，要逐項列出費用明細
3. **處方箋影本** + 藥袋（如有領藥）

### 回台後申請理賠
1. **在意外發生 30 天內**通知保險公司提出理賠申請
2. 備齊：診斷書、收據正本、處方箋、護照影本、登機證 / 入出境紀錄、保單號碼、銀行存摺影本
3. 保險公司一般 7-14 個工作天審核完成
4. **健保自墊核退**：自付醫療費若是急診、住院，回台 6 個月內可向健保署申請部分核退（金額不高但聊勝於無）

### 重要提醒
- **所有單據必須英文版**（韓文版回台幾乎不會被理賠，當場索取最方便）
- 急診請保險公司全程**通話留紀錄**（用 LINE / email 也可）
- 直付醫院 + 自費後核退兩種模式都行，金額大優先選直付


## E5. 插座規格

- **韓國電壓**：220V / 60Hz
- **插座類型**：C 型（兩個小圓孔）+ F 型（兩個圓孔 + 接地片，新建築為主）
- **台灣電壓**：110V / 60Hz → **不相容**

### 必帶：歐規圓頭轉接頭
- 大創、機場、PChome 都有，約 NT$50-150。建議帶 2-3 個（朋友團多人共用）
- **韓國的插座孔徑 4.8mm**：買轉接頭時注意插腳粗度，太細（4mm）會鬆脫
- 韓國當地大創（Daiso）也有賣，約 ₩5,000

### 充電器電壓檢查（重要）
查看充電器標示「INPUT: 100-240V」者 = 全球通用，**只需轉接頭，無需變壓器**
- ✅ 手機、筆電、相機、行動電源、蘋果產品、Switch → 100-240V，安全
- ⚠️ 台灣家用吹風機、捲髮棒、電熱毯 → 多數 110V only，**接到韓國插座會燒掉**
- 需用吹風機 → 飯店都有提供，或買「國際雙電壓」款

### 變壓器
若一定要帶 110V 電器，需另購變壓器（不是只有轉接頭），重量大、成本高，**不建議攜帶**


## E6. 小費文化

**核心原則：韓國沒有小費文化，不要給。**

| 場景 | 是否給 | 備註 |
|---|---|---|
| 餐廳 | ❌ 不需要 | 菜單價含服務費 |
| 計程車 | ❌ 不需要 | 跳表價即實付，零錢說 「괜찮아요 (沒關係)」對方會收下 |
| 飯店 | ❌ 不需要 | 行李員 / 打掃阿姨皆無小費習慣 |
| 高檔燒肉店店員幫剪肉 / 烤肉 | △ 視情況 | 部分長輩會包 ₩10,000 致謝，**非必要** |
| 咖啡店「Tip Jar」 | ❌ 不要丟 | 韓國人 2023 後反小費風氣高漲，本地人也不丟 |
| 美容院、按摩店 | ❌ 不需要 | 技師不期待小費 |

實在感謝對方就**鞠躬 + 說「감사합니다 (謝謝)」**，韓國文化裡這比給錢更得體。


## E7. 廁所資訊

### 哪裡找廁所
- **地鐵站內**：免費、無密碼、乾淨；釜山地鐵每站都有
- **百貨公司 / 大型商場**：樂天、新世界、Centum City、現代百貨，免費 + 高級
- **大學、飯店大廳、星巴克 / 連鎖咖啡店**：免費
- **觀光景點公廁**：海雲台、廣安里沙灘、甘川洞、釜山塔皆有公廁
- **NAVER Map 搜「화장실」(廁所)** 找最近免費廁所

### 衛生紙規則
- **多數新蓋的地鐵站、百貨公司**：可直接丟馬桶（韓國管路升級後 OK）
- **舊建築 / 餐廳 / 小店**：旁邊有垃圾桶就丟桶內，避免堵塞
- **隔間內可能沒衛生紙** → 進廁所前先在門口公用衛生紙台撕一段帶進去
- **隨身帶面紙包**最保險（夜市、傳統市場攤位旁的廁所常無紙）

### 偷拍防範（女性務必注意）
- 韓國公廁偷拍曾是嚴重問題。**進廁所先檢查**：
  - 衛生紙捲筒底部、牆上小孔、鎖頭附近、衣帽鉤、馬桶座下螺絲帽
- 看到可疑小孔 → 撕衛生紙塞住或貼住再使用
- 近年地鐵站、百貨已大幅整治，但小巷餐廳、汽車旅館仍要小心

### 如廁韓文小抄
- 廁所：화장실 (hwajangsil)
- 男 / 女：남자 / 여자
- 衛生紙：휴지 (hyuji)


## H1. 韓國非強制旅遊保險（建議辦理）

- 韓國**不強制**外國觀光客投保（不像申根 / 古巴會在入境查保險）
- 但韓國**外國人就醫不享健保**，自費高（急診 ₩100,000-300,000、住院日 ₩500,000+、手術可達 ₩5,000,000+）
- **強烈建議投保旅平險 + 海外突發疾病醫療**
- 行程不便險（班機延誤、行李遺失）也建議加，韓國 9 月颱風尾巴偶有航班受影響

## H2. 醫療保障建議 ≥ NT$200 萬

| 項目 | 最低建議 | 推薦 |
|---|---|---|
| 旅平險（意外死殘） | NT$500 萬 | NT$1,000 萬 |
| 傷害醫療 | NT$50 萬 | NT$100 萬 |
| **海外突發疾病醫療** | **NT$200 萬** | NT$300 萬 |
| 海外急難救助 / 醫療專機 | 含 | 含 |
| 行程延誤 | NT$5,000 起賠 | NT$10,000+ |
| 行李遺失 | NT$30,000 | NT$50,000 |

5 天 6 夜韓國行的旅平險方案保費約 **NT$300-600 / 人**（含醫療 + 不便險），不貴。

## H3. 除外條款檢查

投保前一定要看清，**這些不賠**：
1. **既有疾病**（180 天內接受過該疾病治療 → 算帶病投保，韓國突發疾病拒賠）
2. **酒醉受傷**（酒測超標 / 喝酒後受傷的醫療費）
3. **危險活動**：滑水、跳傘、潛水、滑雪未加保延伸條款
4. **戰爭、暴動、核災**
5. **整型 / 美容 / 牙科常規治療**（除非急性外傷）
6. **懷孕相關**（多數方案除外）
7. **故意自傷、犯罪行為**

韓國行程要特別注意：滑雪季（12-3 月）、汗蒸幕滑倒、爬山扭傷都會被歸類為一般意外，**只要不是醉酒就 OK**。

## H4. 醫療直付醫院網絡

- **富邦產險**：與 UIA 聯合國際救援合作，南韓含緊急醫療專機轉送，但**直付醫院網絡較有限**，建議當場先付、回台核退
- **國泰產險**：合作 AIG / Allianz，南韓主要大學醫院（首爾延世、釜山大學醫院）多在直付名單
- **新光產險**、**安達產險**：類似配置，可洽客服取得最新韓國直付醫院清單

實務建議：
1. 出發前 email 保險公司問**「釜山 + 大邱直付醫院清單」**
2. 把直付醫院的中英文名 + 地址存手機（緊急時直接給司機看）
3. 若直付醫院太遠 → 先**自費就近治療**，留好收據回台核退
4. **無論直付或自費，先打保險公司 24h 急難專線**取得指引

## H5. 報案 / 理賠文件要求

### 醫療理賠（重複參見 E4）
1. 英文診斷證明書（진단서）
2. 英文收據正本（영수증，含明細）
3. 處方箋影本 + 藥袋
4. 護照 + 登機證 + 行程證明
5. 保單號碼 + 銀行存摺影本
6. **30 天內**提出申請

### 行李 / 班機延誤
1. **班機延誤**：航空公司開的延誤證明（Delay Certificate），**4-6 小時起賠**（看保單）
2. **行李延誤**：行李未到時當場到機場服務台填 PIR (Property Irregularity Report)
3. **行李遺失 / 物品被竊**：到當地警局報案取得**英文報案三聯單**，遺失物品清單建議先列拍照

### 報案重要
- 釜山警察報案打 **112**，可請 1330 三方通話翻譯
- 報案後請警察開「**Police Report**」（中文：警方報案證明），保險公司必收
- 護照遺失 → 1. 警察局報案 2. 駐釜山辦事處補辦緊急護照 / 入國證明書
